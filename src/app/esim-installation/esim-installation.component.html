Installation Guide

How To Install ESimExpress Mobile eSIM

For iPhone running iOS 13. The actual texts or screen might be different depending on the iOS version and the device.

For iPad
  • 01

    Tap Cellular or Mobile Data from Settings

    Make sure your device is connected to a stable WiFi network.

  • 02

    Tap Add Cellular Plan

  • 03

    Scan QR code we provide

    * Internet connection required In case you have only one device and cannot scan the QR code, enter the activation code and SM-DP+ address manually.

    They are provided along with the QR code via email.

  • 04

    Tap Add Cellular Plan

    Note that you cannot go back or cancel after you tap it.

  • 05

    Enter "ESimExpress Mobile" to label your new cellular plan

    You can change labels later in Settings.

  • 06

    Tap Update on Carrier Settings Update

    APN (Access Point Name), which is required for Personal Hotspot, will be configured automatically by doing this step.

  • 07

    Choose Primary as your Default Line

  • 08

    Choose Primary for iMessage, FaceTime and Apple ID

    Or simply tap Continue without selecting any

  • 09

    Choose your default line for cellular data

    Select Primary unless you want to use ESimExpress Mobile eSIM immediately for cellular data.

How To Use ESimExpress Mobile eSIM For Mobile Data

Follow these steps after you arrive at your travel destination.

  • 01

    Turn on ESimExpress Mobile eSIM (Cellular Plan)

    Go to Settings > Cellular or Mobile Data and make sure ESimExpress Mobile eSIM is turned on.

  • 02

    Set it up for Cellular Data and turn Data Roaming on

    Tap Cellular Data and assign ESimExpress Mobile eSIM. Also, enable Data Roaming.

  • OPTIONAL

    For Personal Hotspot, set up APN (Access Point Name) In case you did not update the Carrier Settings (Step 06 above) and want to use Personal Hotspot feature, tap Cellular Data Network and enter "ESimExpress" in all 4 APN fields (Cellular Data, LTE setup, MMS and Personal Hotspot)

SET UP GUIDE FOR iPad

How To Install ESimExpress Mobile eSIM

The instruction below is for iPad (7th generation WiFi+Cellular model) running iPadOS 13. The actual texts or screen might be different depending on the OS version and the device.

  • 01

    Tap Cellular Data > Add a New Plan from Settings

    Make sure your device is connected to a stable WiFi network.

  • 02

    Select "Other …" at the bottom of the carrier list

  • 03

    Scan QR code we provide

    * Internet connection required In case you have only one device and cannot scan the QR code, enter the activation code and SM-DP+ address manually. They are provided along with the QR code via email.

  • 04

    Tap Add Cellular Plan

    Note that you cannot go back or cancel after you tap it.

  • 05

    Make sure it’s added to Cellular Plans

    Some iPadOS versions force to change the plan name from ESimExpress Mobile to Carrier or Carrier-Carrier but it does not affect the eSIM functions.

  • 06

    Tap Update on Carrier Settings Update

    APN (Access Point Name), which is required for Personal Hotspot, will be configured automatically by doing this step.

How To Use ESimExpress Mobile eSIM For Mobile Data

Follow these steps after you arrive at your travel destination.

  • Make sure Roaming is turned on. Select and activate the ESimExpress Mobile eSIM under Cellular Plans.

  • OPTIONAL

    For Personal Hotspot, set up APN (Access Point Name)

    In case you did not update the Carrier Settings (Step 06 above) and want to use Personal Hotspot feature, tap Cellular Data Network and enter "ESimExpress" in all 4 APN fields (Cellular Data, LTE setup, MMS and Personal Hotspot)

How To Install ESimExpress Mobile eSIM

For Pixel 3a running Android 10. Please make sure your device’s Android version and SIM Manager are up-to-date.

  • 01

    Go to settings and tap on connections

  • 02

    Tap on SIM card manager

  • 03

    Tap on Add an eSIM mobile plan

  • 04

    Scan carrier QR code

  • 05

    Tap Activate to add the eSIM to your device

  • 06

    Choose a number (SIM) to use

How To Use ESimExpress Mobile eSIM For Mobile Data

Follow these steps after you arrive at your travel destination.

  • 01

    Under Mobile network, tap ESimExpress Mobile (Downloaded SIM) and activate it.

  • 02

    Make sure Mobile data and Roaming are both turned on. Then scroll down and tap Advanced.

  • 03

    Tap Access Point Names.

  • 04

    Tap the three dots icon on the top right corner and select New APN.

  • 05

    Enter "ESimExpress Mobile" for Name and "ESimExpress" (all lowercase) for APN.

  • 06

    Tap the three dots icon on the top right corner and hit Save.

  • 07

    Select the APN named ESimExpress Mobile you just created.
